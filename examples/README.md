# mmWM Rice
#### more fully featured example panel scripts for lemonbar
## Requires
- lemonbar-xft
- acpid
- [Remix Icons](https://github.com/Remix-Design/RemixIcon/blob/master/fonts/remixicon.ttf) for akita script
